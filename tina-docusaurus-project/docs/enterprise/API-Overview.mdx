## title: API Overview

The Bankuity API platform provides a comprehensive suite of financial analysis tools through a RESTful interface. Our services enable developers to access income verification, risk assessment, and lending intelligence powered by advanced machine learning models.

By leveraging bank statement data, Bankuity transforms raw transaction history into actionable insights, allowing for automated financial decision-making with high confidence.

## Available Services

Our API resources are organized into four primary pillars to support the end-to-end financial analysis workflow.

| Service                      | Description                                                                                                   |
| :--------------------------- | :------------------------------------------------------------------------------------------------------------ |
| **Authentication**           | Manage secure JWT tokens, verify API health via Ping, and retrieve organizational unit configurations.        |
| **Analysis Services**        | Access core intelligence engines including Income Analysis, Bank Verification, and Post-Onboarding Insights.  |
| **Transaction Intelligence** | Retrieve granular processing results, search historical request IDs, and access detailed transaction history. |
| **Connection Management**    | Monitor bridge connectivity with IBV providers and manually trigger data refreshes for the latest snapshots.  |
